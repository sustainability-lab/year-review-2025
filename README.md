# Sustainability Lab - Year in Review 2025

LinkedIn poster for the Sustainability Lab at IIT Gandhinagar.

## Files

- `sustainability-lab-2025-review.html` - Main poster (open in browser to preview)
- `sustainability-lab-2025.png` - Exported PNG (2400x3000px, retina quality)
- `linkedin-post.md` - LinkedIn post text options
- `alum.jpeg` - Alumni photo (local)
- `capture.js` - Script to generate PNG from HTML

## Preview

```bash
open sustainability-lab-2025-review.html
```

## Generate PNG

```bash
npm install
npm run capture
```

## Edit

1. Edit `sustainability-lab-2025-review.html` in any text editor
2. Preview changes in browser
3. Run `npm run capture` to regenerate PNG
